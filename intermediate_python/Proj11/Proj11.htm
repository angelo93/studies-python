PROGRAM SPECIFICATIONS

The zip file that you downloaded contains a file named Stock01.csv. It also contains a file named Proj11YourName.html.

Create a Jupyter notebook named Proj11YourName.ipynb where YourName is actually your own name. Store the file named Stock01.csv in the same folder as the file named Proj11YourName.ipynb.

Your notebook must begin by displaying the following certification (without the quotes).

"I certify that this project is my own work and is not the work of others. I agree not to share my solution with others. - Print your name here."

Then your notebook must replicate the contents of the given file named Proj11YourName.html. Note that the file named Proj11YourName.html was modified to hide the code that you must insert into your ipynb file in order to produce the required tables and graphs.

The format of the tables may be different provided that they display matching data.

Variations among different versions of Jupyter Notebook may result in cosmetic formatting differences in the graphs That will be acceptable so long as the curves, the labels, and the legends are correct.

Download an HTML version of your notebook named Proj11YourName.html where YourName is your actual name.

Encapsulate a copy of the HTML file, a copy of the ipynb file, and a copy of the file named Stock01.csv in a zip file and submit the zip file via Blackboard.

Be sure that your actual name appears in the certification and in the file names as indicated.
