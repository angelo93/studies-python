PROGRAM SPECIFICATIONS

The zip file that you download for this assignment will contain a file named Proj01Output.html. Open that file in your browser and examine it very carefully.

Create a Jupyter Notebook named Proj01YourName.ipynb which, when downloaded as an HTML file, matches the file named Proj01Output.html. Match the instructional text along with the results of those instructions.

Encapsulate a copy of your HTML output file and a copy of the ipynb file in a zip file and submit the zip file via Blackboard.

As a result of the differing behavior of different versions of Jupyter Notebook, you may not be able to match the cosmetic aspects of the file named Proj01Output.html in every detail. For example, the text need not be blue and need not be underlined. However, the headers must be anchors to the links at the top of the page and the links at the top must be links to the headers.

Don't overlook the requirement to create a link to an external website and the requirement to create a link to an internal anchor.

Also don't overlook the requirement to match the text under the header titled Escape characters and inline monospace.

Be sure that your name appears in the certification as indicated.
